# PFAS Multipollutant Methods & Immune Biomarkers – Master's Thesis

This repository contains the R code and simulations used in my master’s thesis:  
**"Evaluating Multipollutant Methods for PFAS Exposure and Immune Outcomes in Adolescents"**  
[University Name], [Program], 2025

---

## Project Overview

### 1. Applied Data Analysis  
- Study associations between multiple PFAS compounds measured in serum and immune biomarkers, including:  
  - Leukocyte count  
  - CD4⁺ and CD8⁺ T cells and their ratio  
  - Asthma and eczema prevalence  
- Use multipollutant regression methods to estimate individual and joint effects, accounting for correlated exposures.

### 2. Simulation Study  
- Evaluate performance of statistical methods for mixtures, focusing on:  
  - Quantizing exposures (categorizing by quantiles)  
  - OLS, penalized regressions (Lasso, Elastic Net)  
  - Weighted Quantile Sum (WQS) and Quantile G-Computation (qgcomp)  
  - Bayesian variable selection and kernel machine regression  
- Metrics include bias, RMSE, coverage, and R² for joint effect estimates.
